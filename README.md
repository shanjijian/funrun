# FunRun

通过规则过滤简单的无效的告警。

地址：https://github.com/shanjijian/funrun

## 架构

前端：`vue3` + `axios` + `ElementPlus` 

后端：`fastapi`  + `RestrictedPython`  + `PyYAML`
